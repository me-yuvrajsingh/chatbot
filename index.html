ChatBot (Made for SciAstra.com)

Made by: Yuvraj Singh

About Chatbot:

  • This chatbot is made using pure HTML, CSS and Js
  • It is linked with SciAstra webpages
  • It can respond to various commands
  • The color scheme used in this chatbot has been inspired by the SciAstra website design.
  • This is not responsive yet
  • The current version of the chatbot includes only basic queries, but it can be upgraded further

Some Queries it can answer:

  • Hello
  • How are you?
  • Who are you?
  • What is your name
  • Navigate me to the blog page
  • Navigate me to contact page
  • What is the date today
  • How to contact you
  • What is your contact number
  • What is your email id
  • Where is your office located
